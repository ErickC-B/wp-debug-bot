# WP Debug Bot – Diagnóstico instantâneo de WordPress

**Projeto criado em novembro/2025 com o intuito de testar e aprender o uso da API da OpenAI**

## O que esse bot faz?

Recebe qualquer erro de WordPress > devolve diagnóstico completo + solução passo a passo em alguns segundos.

Perfeito para suporte técnico de quem usa WordPress!

```

cd wp-debug-bot
pip install -r requirements.txt
streamlit run app.py
```
